== Description

  Special Thanks Page Generator

== Features

  Generate HTML or JSON, YAML, CSV from your Gemfile

== Examples

  $ cd path/to/your_project
  $ bundle
  $ thanks g bootstrap
        create  thanks
        create  thanks/index.html

== Requirements

 Ruby 1.9+ 

== Install

  $ gem install arigato
